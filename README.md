City Pulse - Local Events Explorer

Overview:
The app includes features from the Ticketmaster API. Users can search events using a keyword and city, and view details of each event.

Features:

* Splash Screen
* User can create an account using email and password through Firebase
* User can log in using their registered email and password
* Search events by keyword and city
* View event list and detailed event information
* Event detail page includes basic info and a location marker on the map
* Favorite and unfavorite events, stored locally using AsyncStorage
* Switch between English and Arabic languages
* Basic user profile showing user name, email, list of favorite events, and logout option

Setup:

1. Clone the repository
2. Install dependencies using yarn or npm
3. Run the app using:

   * yarn run ios
